# sa-flex
Towards robust object discovery in the diverse visual complexity: Flexible Slot-Attention

### Installation
``` bash
pip install -r requirements.txt
```

### How to Run
``` bash
python scripts/train.py
# change configurations in src/slot_attention/params.py 
```
